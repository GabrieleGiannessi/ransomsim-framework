# SEC4H project - ransomware attack scenario

## Requirements

This project uses [`uv`](https://docs.astral.sh/uv/) for dependency management.

If you don't have it installed, follow the [official installation guide](https://docs.astral.sh/uv/).

## Installation

Once `uv` is installed, sync the project dependencies by running:

```bash
uv sync
```

## Running the Application

To start both the API and the frontend application, run:

```bash
make all
```
